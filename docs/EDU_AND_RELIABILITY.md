# OpenAI and .edu perks

The app uses **OpenAI (GPT-4o-mini)** only for summary and quiz generation.

---

## .edu / education perks (OpenAI)

- **ChatGPT Edu**: For institutions (shared credit pool). Mainly the ChatGPT product; contact OpenAI for API-related terms.
- **API**: Contact **OpenAI sales** for **Education / Edu plan** or **API pricing for .edu**. Mention your .edu domain and study-assistant use case.
- **Researcher Access Program**: Up to **$1,000 API credits** for academic researchers. Application windows (e.g. March, June). Apply with the same account you use for the API; credits apply to that account.

---

## What you need in .env

- **OPENAI_API_KEY** — required (create at https://platform.openai.com/api-keys).
- **Supabase** (recommended for production persistence) — `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`, and optional `SUPABASE_STORAGE_BUCKET`.
